How to install HP_autologin userscript

1. Install a UserScript manager (GreaseMonkey for Mozilla Firefox, TamperMonkey for Google Chrome or any other UserScript manager, really)
2. Download the script here (You are actually supporting me, the creator, while using this link. Thanks !)
3. Follow the instructions on the UserScript manager to install it. This should only be a drag-and-drop operation ;-)

How to install HP_autologin userscript

1. Install a UserScript manager (GreaseMonkey for Mozilla Firefox, TamperMonkey for Google Chrome or any other manager, really)
2. Download the script here (You are actually supporting me, the creator, while using this link. Thanks !)
3. Follow the instructions on the UserScript manager to install it. This should only be a drag-and-drop operation ;-)

You can also use this bookmarklet : Hyperplanning (with AutoLogin) (drag it, drop it in your bookmarks bar, and don't forget to edit it with your credentials !)
